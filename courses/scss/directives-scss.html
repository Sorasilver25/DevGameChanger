Les directives @ en SCSS

Apprenez à utiliser les directives SCSS pour organiser, factoriser et dynamiser votre code CSS.

Dernière mise à jour : 30.08.2025

Introduction

SCSS propose une série de directives puissantes qui permettent d'organiser, de factoriser et de dynamiser votre code CSS. Voici les plus courantes :

@each – Boucle sur une liste

Permet d'itérer sur une liste ou une map de valeurs. 

$colors: red, green, blue;

@each $color in $colors {
  .text-#{$color} {
    color: $color;
  }
}

Génère :

.text-red   { color: red; }
.text-green { color: green; }
.text-blue  { color: blue; }

@mixin & @include – Fonctions réutilisables

Un mixin permet de définir un bloc de code réutilisable avec ou sans paramètres. 

@mixin rounded($radius) {
  border-radius: $radius;
  -webkit-border-radius: $radius;
}

.card {
  @include rounded(12px);
}

Génère :

.card {
  border-radius: 12px;
  -webkit-border-radius: 12px;
}

@if / @else – Conditions logiques

Permet d'appliquer des styles en fonction de conditions. 

$theme: dark;

body {
  @if $theme == dark {
    background: #111;
    color: #eee;
  } @else {
    background: #fff;
    color: #111;
  }
}

Résumé des directives

  • @each : boucle sur liste
  • @mixin / @include : fonctions personnalisées
  • @if / @else : conditions logiques
  • @media : responsive design
  • @extend : héritage de style
