Préprocesseur CSS – Sass / SCSS

Découvrez comment utiliser Sass/SCSS pour créer des styles maintenables et dynamiques à l'aide de l'interpolation de variables.

Dernière mise à jour : 25.08.2025

Introduction

Sass (Syntactically Awesome Stylesheets) est un préprocesseur CSS qui permet d'écrire des feuilles de style plus maintenables et dynamiques grâce à des variables, fonctions, boucles, héritages, etc.

Dans sa syntaxe la plus répandue (SCSS), il est compatible avec la syntaxe CSS classique.

Interpolation avec #{$variable}

Le #{...} en SCSS permet d'interpoler une variable Sass dans une chaîne. Cela est particulièrement utile pour :

  • Créer dynamiquement des noms de classes ou d'identifiants
  • Générer plusieurs propriétés CSS ou sélecteurs

Exemples d'interpolation

Exemple 1 – Interpolation dans un nom de classe

$theme: dark;

.button-#{$theme} {
  background-color: black;
  color: white;
}

Génère :

.button-dark {
  background-color: black;
  color: white;
}

Exemple 2 – Interpolation dans une propriété

$side: left;

.margin-#{$side} {
  margin-#{$side}: 20px;
}

Génère :

.margin-left {
  margin-left: 20px;
}

Exemple 3 – Boucle + interpolation

$themes: light, dark, blue;

@each $theme in $themes {
  .btn-#{$theme} {
    background: url('/img/#{$theme}.png');
  }
}

Génère :

.btn-light {
  background: url('/img/light.png');
}
.btn-dark {
  background: url('/img/dark.png');
}
.btn-blue {
  background: url('/img/blue.png');
}

Résumé

  • #{$var} permet d'injecter dynamiquement une valeur dans une chaîne
  • Utile dans les noms de classes, sélecteurs, propriétés CSS
  • Utilisable dans les boucles et les mixins

Cette fonctionnalité rend SCSS très puissant pour créer des styles dynamiques et évolutifs.
