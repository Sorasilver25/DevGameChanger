Requêtes SQL

Guide complet des requêtes SQL : de la sélection simple aux requêtes avancées avec jointures, agrégations et sous-requêtes

Dernière mise à jour : 13.11.2025

Requêtes de base

Les requêtes SELECT permettent de récupérer des données d'une ou plusieurs tables.

Sélection simple

-- Sélectionne une colonne spécifique
SELECT `Immatriculation` FROM `Voiture`;

Sélection avec condition WHERE

-- Sélectionne toutes les marques des voitures de 1996
SELECT `Marque` FROM `Voiture` WHERE `Année` = 1996;

-- Sélectionne toutes les informations des voitures appartenant à 'Id01'
SELECT * FROM `Voiture` WHERE `IdProprio` = 'Id01';

Sélection avec plage de valeurs

-- Sélectionne toutes les voitures dont le prix est entre 10 000 et 20 000
SELECT * FROM `Voiture` 
WHERE `Prix` BETWEEN 10000 AND 20000;

Fonctions d'agrégation

Les fonctions d'agrégation permettent d'effectuer des calculs sur un ensemble de lignes.

COUNT - Compter les enregistrements

-- Nombre total de voitures
SELECT COUNT(*) FROM `Voiture`;

-- Nombre de voitures de marque 'Fiat'
SELECT COUNT(`Marque`) FROM `Voiture` 
WHERE `Marque` = 'Fiat';

AVG et SUM - Moyenne et somme

-- Moyenne et somme des prix des voitures
SELECT AVG(`Prix`) AS PrixMoyen, 
       SUM(`Prix`) AS PrixTotal 
FROM `Voiture`;

MIN et MAX - Valeurs minimales et maximales

-- Prix minimum et maximum
SELECT MIN(`Prix`) AS PrixMin, 
       MAX(`Prix`) AS PrixMax 
FROM `Voiture`;

Regroupement avec GROUP BY

GROUP BY permet de regrouper les résultats selon une ou plusieurs colonnes et d'appliquer des fonctions d'agrégation sur chaque groupe.

Regroupement simple

-- Compte le nombre de voitures par propriétaire
SELECT `IdProprio`, COUNT(*) AS NombreVoitures
FROM Voiture 
GROUP BY `IdProprio`;

Regroupement avec HAVING

-- Sélectionne la somme des prix des voitures par propriétaire
-- uniquement pour ceux dont le total dépasse 50 000
SELECT IdProprio, SUM(Prix) AS TotalPrix
FROM `Voiture`
GROUP BY IdProprio
HAVING SUM(Prix) > 50000;
Note : HAVING filtre après le regroupement, tandis que WHERE filtre avant.

Sous-requêtes

Les sous-requêtes permettent d'utiliser le résultat d'une requête dans une autre requête.

Sous-requête avec comparaison

-- Sélectionne toutes les informations de la voiture la plus chère
SELECT * FROM Voiture 
WHERE `Prix` = (SELECT MAX(`Prix`) FROM Voiture);

Sous-requête avec IN

-- Liste des personnes qui n'ont pas de voiture
SELECT Nom, Prénom FROM Personnes
WHERE IdProprio NOT IN (
  SELECT DISTINCT IdProprio FROM Voiture
);

Sous-requête avec EXISTS

-- Propriétaires qui possèdent au moins une voiture chère
SELECT DISTINCT `IdProprio` FROM `Voiture` v1
WHERE EXISTS (
  SELECT 1 FROM Voiture v2 
  WHERE v2.IdProprio = v1.IdProprio 
  AND v2.Prix > 20000
);

Jointures (JOIN)

Les jointures permettent de combiner les données de plusieurs tables basées sur une relation.

INNER JOIN

-- Sélectionne le propriétaire avec le plus grand nombre de voitures
SELECT p.Nom, p.Prénom, COUNT(*) AS NombreVoitures
FROM Personnes p
INNER JOIN Voiture v ON p.IdProprio = v.IdProprio
GROUP BY p.Nom, p.Prénom
HAVING COUNT(*) >= ALL(
  SELECT COUNT(*) FROM Voiture GROUP BY IdProprio
);

LEFT JOIN

-- Liste toutes les personnes et leurs voitures (même sans voiture)
SELECT p.Nom, p.Prénom, v.Marque, v.Prix
FROM Personnes p
LEFT JOIN Voiture v ON p.IdProprio = v.IdProprio;

Jointure multiple

-- Jointure entre 3 tables
SELECT c.NomClient, v.Marque, a.DateAchat
FROM Clients c
INNER JOIN Achats a ON c.IdClient = a.IdClient
INNER JOIN Voiture v ON a.IdVoiture = v.IdVoiture;

Opérateurs avancés

DISTINCT - Valeurs uniques

-- Sélectionne les propriétaires uniques qui possèdent des voitures chères
SELECT DISTINCT `IdProprio` FROM `Voiture` 
WHERE `Prix` > 20000;

LIKE - Recherche par motif

-- Recherche les voitures dont la marque commence par 'F'
SELECT * FROM Voiture WHERE Marque LIKE 'F%';

-- Recherche les voitures avec 'iat' dans la marque
SELECT * FROM Voiture WHERE Marque LIKE '%iat%';

ORDER BY - Tri des résultats

-- Trie par prix décroissant
SELECT * FROM Voiture ORDER BY Prix DESC;

-- Tri multiple
SELECT * FROM Voiture 
ORDER BY Marque ASC, Prix DESC;

LIMIT - Limiter le nombre de résultats

-- Les 5 voitures les plus chères
SELECT * FROM Voiture 
ORDER BY Prix DESC 
LIMIT 5;

Manipulation de données

INSERT - Ajouter des données

-- Insertion d'une nouvelle voiture
INSERT INTO Voiture (Immatriculation, Marque, Année, Prix, IdProprio)
VALUES ('AB-123-CD', 'Renault', 2023, 25000, 'Id01');

UPDATE - Modifier des données

-- Mise à jour du prix d'une voiture
UPDATE Voiture 
SET Prix = 18000 
WHERE Immatriculation = 'AB-123-CD';

DELETE - Supprimer des données

-- Suppression d'une voiture
DELETE FROM Voiture 
WHERE Immatriculation = 'AB-123-CD';

Commande Description Exemple
SELECT Récupère des données SELECT * FROM table
WHERE Filtre les résultats WHERE Prix > 1000
GROUP BY Regroupe les résultats GROUP BY Marque
HAVING Filtre après regroupement HAVING COUNT(*) > 5
JOIN Combine plusieurs tables INNER JOIN ON id = id
ORDER BY Trie les résultats ORDER BY Prix DESC
LIMIT Limite le nombre de résultats LIMIT 10
COUNT() Compte les enregistrements COUNT(*)
SUM() Calcule la somme SUM(Prix)
